This is a crazy looking font!

I implemented this font using inline stylesheets where I imported a font style from a URL taken from google font API, It is easy to find and quickly incorporate font's in websites using this process or even moving the import to external stylesheets as needed. After this I used inline stylesheeting to further utilize the imported fonts by specifying the header to use that font family and style.
